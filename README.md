# Sridhaa Visuals - Premium Photography Portfolio

A luxury cinematic portfolio website for photographers and video editors, built with React, Tailwind CSS, and Framer Motion.

## 🎨 Features

- **Fullscreen Hero Section** - Cinematic background with smooth animations
- **About Section** - Professional profile with stats and expertise showcase
- **Photography Gallery** - Masonry layout with category filtering
  - Pre-wedding
  - Weddings
  - Portraits
  - Temple photography
  - Events
- **Video Showcase** - Hover preview effects and video modal
- **Testimonials** - Client reviews with elegant design
- **Social Media Integration** - Instagram feed and social links
- **Contact Form** - Booking system with validation
- **Responsive Design** - Mobile-first approach
- **Smooth Animations** - Framer Motion powered transitions
- **Luxury Theme** - Black and gold aesthetic with premium spacing

## 🚀 Tech Stack

- **React 18** - UI library
- **Vite** - Build tool
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Animation library
- **React Icons** - Icon library

## 📦 Installation

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```

## 🎨 Customization

### Colors

Edit `tailwind.config.js` to customize the color palette:

```js
colors: {
  gold: { ... },
  luxury: { ... }
}
```

### Content

Update the following in component files:
- Images: Replace Unsplash URLs with your own
- Text: Modify titles, descriptions, and copy
- Contact info: Update email, phone, and location
- Social links: Add your social media URLs

### Fonts

Current fonts (from Google Fonts):
- **Headings**: Playfair Display
- **Body**: Inter
- **Code**: Fira Code
